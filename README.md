# Cadence VLSI Design Labs

This repository contains my Cadence Virtuoso lab work and project reports for CPEG660. The work starts with basic Cadence setup and tutorials, then progresses through schematic design, simulation, layout, DRC/LVS verification, and full custom digital circuit implementation.

## Contents

### Cadence Basics and Tutorials

These files document the initial learning exercises used to understand the Cadence design flow:

- `Using_Cadence.pdf` — Cadence setup and library manager basics
- `Inverter.pdf` — CMOS inverter schematic and symbol creation
- `TestBench.pdf` — Inverter testbench creation and transient simulation
- `ParametricAnalysis.pdf` — DC simulation and transistor sizing using parametric analysis
- `DRCandLVSchecks.pdf` — Inverter layout with DRC and LVS verification
- `NANDgate.pdf` — Two-input NAND gate schematic, layout, DRC, and LVS

### Project 1: MUX Design

Project 1 focuses on multiplexer design and verification in Cadence. The work includes schematic creation, simulation, layout, extraction, DRC, and LVS checks.

Files:

- `2-1MUX.pdf`
- `2-1MUXlayout.pdf`
- `2-1MUXsimulation.pdf`
- `2-1MUXcheck.pdf`

### Project 2: 4-bit Carry Look-Ahead Adder

Project 2 is the final project: a transistor-level implementation of a 4-bit Carry Look-Ahead Adder (CLA). The design uses generate and propagate logic to compute carry signals efficiently and includes schematic design, layout, simulation, and DRC/LVS verification.

Files:

- `4bitCLAadder_schematic.pdf`
- `4bitCLAadder_layout.pdf`
- `4bitCLAadderFINAL.pdf`

## Tools Used
